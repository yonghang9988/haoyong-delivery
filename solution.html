豪勇系统产品方案

面向中小型配送企业的订单、仓储、分拣、配送、考勤、记账一体化管理平台。
进入PC后台 进入司机端
效率目标分拣效率 +50%
时效目标订单处理 -30%
库存目标准确率 99%+
可用性99.5%

业务架构

小程序 + App/PDA + PC后台

客户下单 订单审核 采购补货 扫码分拣 配送签收 库存更新 费用结算 经营报表

进销存管理

采购入库、销售出库、实时库存、库位管理、库存预警、盘点差异。

分拣管理

按单/波次分拣、扫码校验、蓝牙秤称重、标签打印、异常上报。

配送管理

智能派单、路线规划、装车确认、在途跟踪、拍照签收、异常处理。

考勤管理

GPS/WiFi/人脸打卡、排班、补签、加班统计、绩效关联。

记账管理

仓储费、操作费、配送费自动生成，账单汇总、核销对账、经营报表。

系统管理

商品、仓库、人员、车辆、角色权限、多仓数据隔离、操作日志审计。

角色权限

按角色、仓库、终端隔离数据

老板/管理者经营看板、审批、人员管理
客户下单、查订单、支付、签收确认
库管员入库、盘点、库存预警
分拣员扫码拣货、称重、打包、异常上报
配送员接单、路线导航、拍照签收
财务人员账单、核销、报表导出

技术架构

模块化分层，支持后续真实系统开发

PC：Vue3 + Element Plus 小程序：UniApp App/PDA：UniApp / Flutter 后端：Spring Boot + MyBatis-Plus 数据库：MySQL 8.0 + Redis 实时推送：WebSocket

实施路线

先跑通主流程，再扩展考勤和财务

  1. MVP 2-3个月小程序下单、管理后台、基础进销存、商品/客户/仓库管理。
  2. 第二阶段 2-3个月分拣App/PDA、配送App、标签打印、蓝牙秤对接。
  3. 第三阶段 1-2个月考勤模块、记账模块、费用自动计算与账单。
  4. 第四阶段 1个月数据看板、报表中心、第三方系统集成优化。
